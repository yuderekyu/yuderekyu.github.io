# yuderekyu.github.io
My personal site.

## Getting Started
Want to creep around? Ensure you have node version manager ([nvm](https://github.com/nvm-sh/nvm)) installed. If not, follow their instructions.

### Installation
Install node v14:
```sh
nvm install 14
```

Use node v14:
```sh
nvm use 14
```

Install dependencies:
```sh
npm install
```

### Development
Start the server:
```sh
npm start
```
